# USB_CAN_DJI
## How to
```bash
cd usb_can_dji
source usb_can/bin/activate
python3 usb_can.py
```
User can enter a number from -10 to 10.  
The number corresponds to the current value.  
To stop, use ctrl-c to terminate the process.
